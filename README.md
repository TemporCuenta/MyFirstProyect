# Installation
1. Need to download & install termux from f-droid
2. run this command 
``` 
apt update && apt upgrade -y  
```
3. run 
```
pkg install wget -y
```
4. run 
```
apt install libc++ libnl libpcap libsqlite openssl pcre zlib -y
```
5. Now, we need to know our device architecture.
Run
```
uname -m
```
